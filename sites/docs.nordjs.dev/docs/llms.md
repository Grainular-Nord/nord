# Large Language Model Documentation

Nord provides optimized documentation structures for Large Language Models (LLMs) and AI agents. We adhere to the standard [llms.txt](https://llmstxt.org/) convention to ensure that AI assistants can accurately ingest and understand the documentation without parsing HTML.

The following files are available on the root level of the documentation.

- [`/llms.txt`](/llms.txt): A concise summary of the framework, key concepts, and high-level API reference. Best for quick context.
- [`/llms-full.txt`](/llms-full.txt): The complete documentation, including detailed guides, full API signatures, and examples. Use this for deep-dive tasks.

## Setup for Contributors

For contributors working on documentation, the above files are generated by the [vitepress-plugin-llms](https://github.com/okineadev/vitepress-plugin-llms) from the source markdown in the `docs/` directory. They are served from the public root of this site.
